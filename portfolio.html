Projeto elétrico

Escolinha do Fluminense

Nós da Tesla Engenharia o projeto elétrico da Escolinha do Fluminense em Palmas (TO). O objetivo foi reformar toda a instalação da escolinha para garantir a eficiência e, sobretudo, a segurança dos alunos.

Projeto de medição agrupada

Realizamos o projeto de medição agrupada em um conjunto de kitnets em Palmas (TO). Dentre os serviços inclusos no portfólio da Tesla Engenharia, este é o responsável pelo projeto que irá garantir a regulamentação frente à concessionária de energia local para o fornecimentos a multiplas unidades consumidoras, sendo também necessário para o incio do fornecimento de energia.
